# p8fighter
2D spamming game
Lopez Romain
Thievenaz Hugo
Antherieu Justine

Le projet P8 Fighter est mené par trois étudiants de l’école Polytech Lyon: Justine ANTHERIEU, Romain LOPEZ, et Hugo Thievenaz.
Il est réalisé pour les responsables de l’UE LIFAP4 de l’université Claude Bernard Lyon 1.
P8 Fighter est un jeu de type street fighter, basé sur l’environnement de la classe P8 de la promotion PeiP2 2017.

Les deux personnages principaux sont donc Tancrède Suard et Justine ANTHERIEU, deux élèves de notre classe. Le décor se base sur une image de notre salle de classe habituelle.
Ce jeu se joue à deux joueurs, l’un jouant avec le personnage Justine et l’autre jouant avec le personnage Tancrède.	
Le but est de mettre son adversaire KO, sans l’être soi-même.  Un personnage est KO si sa jauge de vie tombe à zéro.
Pour cela, chaque personnage peut :	
-se déplacer de gauche à droite, 	
-sauter, 
-s’accroupir,	
-attaquer :
      avec le poing 	
      avec le pied
Chaque personnage possède des caractéristiques : vitesse, dégâts infligés par le poing, par le pied jauge de vie.
Des événements apparaîtront aléatoirement. Il peut s’agir :	
-> de Romain : ce personnage invoque des « infus » aléatoires, aux différents effets : ralentir ou accélérer les personnages, infliger des dégâts, soigner.	
-> de Félix : ce personnage crée un rayon lumineux pendant quelques secondes, et si Justine ou Tancrède pénètrent dedans, ils regagnent des points de vie.
-> de Mme. Dejou : ce personnage accélère fortement la vitesse du joueur qui l’attrape.
-> de M. Attal : ce personnage bloque la prochaine attaque portée au joueur qui l’attrape.
